# 图片资源目录

此目录用于存放网站的图片资源。

## 建议的图片

1. **Logo** - 公司Logo图片（建议尺寸：200x60px）
2. **Hero背景图** - 首页背景图片（建议尺寸：1920x1080px）
3. **服务图标** - 各服务项目的图标或图片
4. **团队照片** - 团队成员照片
5. **公司照片** - 公司环境、办公场所等

## 当前状态

网站目前使用emoji图标作为占位符。如需替换为实际图片：

1. 将图片文件放入此目录
2. 在HTML文件中将emoji替换为 `<img>` 标签
3. 例如：`<div class="service-icon">🚗</div>` 替换为 `<img src="assets/images/car-icon.png" alt="车险服务" class="service-icon">`

## 图片优化建议

- 使用WebP格式以获得更好的压缩比
- 图片大小控制在200KB以内
- 为不同设备准备不同尺寸的图片（响应式图片）
